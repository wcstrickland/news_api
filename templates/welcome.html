There are 5 different endpoints:

  1. All articles:
    • http://127.0.0.1:5000/all
  2. Search by keyword:
    • /key/field/keyword
    • example:
    • http://127.0.0.1:5000/key/topic/joe
  3. Search by date range:
    • /daterange/from_date/to_date
    • example:
    • http://127.0.0.1:5000/daterange/2020-10-31/2020-11-10
  4. Search by date range and keyword:
    • /keyrange/from_date/to_date/field/keyword
    • example:
    • http://127.0.0.1:5000/keyrange/2020-10-31/2020-11-10/topic/joe
  5. Search by all and indicate columns for display
    • /specific/column/from_date/to_date/field/keyword
    • example:
    • http://127.0.0.1:5000/specific/headline/2020-10-31/2020-11-10/topic/joe
    • This will produce only the headline from resulting articles
